A collection of iOS components written in Swift.

## Requirements

* iOS 11+
* Swift 5
* Xcode 14+

## Installation

#### [CocoaPods](http://cocoapods.org) (recommended)

```ruby
pod 'FSUIKitSwift'
```

#### [Carthage](https://github.com/Carthage/Carthage)

````bash
github "lifution/FSUIKit"
````
